# Tenhou Visualizer

[ダウンロードページ](https://github.com/CrazyBBB/tenhou-visualizer/releases/)

## 概要

- 天鳳の牌譜を可視化するツール
- 実行するにはJavaの実行環境が必要

![](screenshot.png)

牌画像は[麻雀豆腐](http://majandofu.com/mahjong-images)のものを使用させてもらいました  
シャンテン数の計算は[麻雀C言語プログラム集](http://cmj3.web.fc2.com/)を参考にさせてもらいました
